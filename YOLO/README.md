# CNN

- Հիմնական ֆայլը, որտեղ կատարվում է «Train» և կառուցվում են «plot»-ները, դա՝ `YOLO_PascalVoc12.ipynb` ֆայլն է։  
  -- Ֆայլը ներկա կարգաբերումներով ճիշտ կգործարկվի միայն **Google CoLab**-ում։  
  -- Սկրիպտը գործարկեք հետևյալ հաջորդականությամբ՝  
    1. **Requirements download** *(Եթե սկրիպտը աշխատացնում եք առաջին անգամ և **Google Drive**-ում դեռ չունեք պահպանված տվյալները, ապա գործարկեք այս կտորը նույնպես)*  
    2. **Import from drive\\** *(«ultralytics»-ի և PascalVoc2012-ի ներբեռնում դեպի Google Drive)* 
    3. **RUN** *(Օբյեկտների տեսակների հայտարարում, ձևափոխում դեպի «YOLO» ֆորմատ, «PascalVoc2012» տվյալների բազայից նկարների կիսում՝ «train» և «val)* 
    4. **Train** *(Տվյալ օրինակը ուսուցանված է 45 շրջապտույթ, ուսուցանմատ տվյալները պահպանվում են «yolo_voc_finetune» պանակում, որը գտնվում է՝ «runs/detect/yolo_voc_finetune/»)*
    5. **Loocking for the best hyperparameters** *(աշխատացնել միայն այն դեպքում, եթե ուզում եք գտնել լավագույն հիպերպարամետրերը)*  
    6. **Save `Yolo_Voc_Finetune` to Drive** *(արդյունքների պահպանումը Google Drive-ում՝ արդյունքները չկորցնելու համար)*  
    7. **Detect** *(1 նկարի օրինակի վրա կանխատեսման իրականացում, «accuracy»-հաշվարկ)*  
    8. **PLOT**  
       - 1․ mAP50, precision, recall  
       - 2․ Train loss, Val loss  
    9. **Import `yolo_voc_finetune` from Drive** *(Google Drive-ում նախկինում պահպանված պանակի ներմուծում Google CoLab միջավայր)*
